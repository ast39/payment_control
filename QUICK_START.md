# 🚀 Быстрый старт проекта

## 📦 Установка зависимостей
```bash
make install
```

## 🗄️ Выполнение миграций
```bash
make migrate
```

## 🔧 Запуск бэкенда
```bash
make start-backend
```

## 🎨 Запуск фронтенда
```bash
make start-frontend
```

## 👤 Создание пользователя
```bash
make create-user
```

## 🌐 Доступ к приложению

- **Фронтенд**: http://localhost:3000
- **API (бэкенд)**: http://localhost:3001/api
